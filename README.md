# Random Case

Randomly upper and lower characters in a string

## UX

**Getting Started**

Input text, e.g. `Hello World` in the text field.  As you key in the text, the characters are randomly in upper and lower case.

## Features

Allows the user to enter text in the text field, to see characters randomly change to upper or lower case.

## Technologies

Uses HTML5, CSS3, JavaScript, Bootstrap 5.3.2 and jQuery 3.7.1.

## Testing

To preview, use the command `python3 -m http.server` in the terminal and press `Enter`.

Validator will be performed.

## Deployment

Deployed on [GitHub Pages](https://derektypist.github.io/random-case) at the main branch.
